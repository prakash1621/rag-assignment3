# Fare Source Codes & Precedence

## Order of Precedence
TC → TK → TT → LO → GP → RF → DG → GF → Defaults

## Key Rules
- TC provides coupon-level proration (highest priority)
- TK and TT used when TC is unavailable
- LO fares apply to loyalty bookings (base fare only)
- RF fares used for unticketed bookings with SBR pricing
- GP/DG apply to group bookings
- GF applies to government fares
- Defaults used when no criteria is met
