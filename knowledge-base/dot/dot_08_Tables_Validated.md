# DOT Tables Validated

The following DOT (Digital Operations Transformation) tables were validated as part of the DOT Customer Journey migration:

- CUST_JRNY_TRIP_FLT_LEG_CHG
- CUST_JRNY_TRIP
- CUST_JRNY_TRIP_WTH_FLT

These are the core DOT Customer Journey dataset tables.


 
# DOT Tables Details

1. CUST_JRNY_TRIP_FLT_LEG_CHG: This table contains information about customer journey trip flight leg changes.

2. CUST_JRNY_TRIP: This table stores customer journey trip data.

3. CUST_JRNY_TRIP_WTH_FLT: This table includes customer journey trip details with flight information.
